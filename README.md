FuZFuZ es un sencillo fuzzer creado en python para descubrir posibles directorios web, útil para pruebas de penetración.


#### Instalación:

```
> git clone https://github.com/R3LI4NT/FuZFuZ
> cd FuZFuZ
> python3 fuzfuz.py -h
```

## Uso:
| COMANDO | DESCRIPCION |
| ------------- | ------------- |
| -u/--url | URL objetivo  |
| -w/--wordlist  | Ruta del diccionario a usar  |
