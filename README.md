FuZFuZ es un rápido y sencillo fuzzer creado en python para descubrir posibles directorios web.


#### Instalación:

```
> git clone https://github.com/R3LI4NT/FuZFuZ
> cd FuZFuZ
> python3 fuzfuz.py
```

## Uso:
| COMANDO | DESCRIPCION |
| ------------- | ------------- |
| -u/--url | URL objetivo  |
| -w/--wordlist  | Ruta del diccionario a usar  |
